🚔 Crime Record Management System (CRMS)
📌 Project Description

Crime Record Management System is a terminal-based Core Java project developed using JDBC and MySQL. The system is designed to manage criminal records efficiently through the command line interface. It allows users or administrators to store, update, delete, and search crime-related information digitally.

This project helps reduce manual paperwork 📄 and improves record management using database connectivity 🔗.
